# Logistic Regression

## Sigmoid Function

$$\sigma(x) = \frac{1}{1+e^{-x}}=\frac{e^x}{1+e^x}$$

<img src="https://github.com/yuchentang/yuchentang.github.io/blob/main/assets/images/sigmoid.png?raw=true" alt="sigmoid" width="500"/>

## Logistic Regression Model

$$\hat{y}=\sigma(w^Tx+b)$$
