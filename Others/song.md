# 《滇东幻想症》By 本能实业 DEMO

<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=86 src="//music.163.com/outchain/player?type=2&id=1890709566&auto=1&height=66"></iframe>

这首歌的元素缭乱到堪称炫技的地步，但又能逐个击中我。你可以谈狂热又本能的个人崇拜，可以谈互不否定的前后两个30年，可以谈忆秦娥。可以谈我最爱的基于理性人假设和效用的微观经济学甚至计划经济。谈我学习机器学习和推荐系统时初探信息茧房和数据隐私的感受。可以谈我喜爱的哈耶克，朗诵他的通往地狱的道路往往是由善意铺就的。生活的海洋，对我来说，是小我的海洋，人作为一个个体而具有意义的海洋。我相信人的小我是有意义且重要的，我也希望它能席卷而来。如此看来，我还是得靠幻想而活，无论是爱情还是生活。
