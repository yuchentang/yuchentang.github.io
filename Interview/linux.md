# Linux基本命令

## `ls`
```sh
-l 列出目录或者文件的详细信息。比如权限、修改时间等等
-a 列出当前目录下所有文件，包括隐藏文件（已点开头的都是隐藏文件）
```

## `cd`

change directory, 变更目录

```sh
cd <directory path>
cd ../ 切换到上级目录
cd / 切换到根目录
cd ~ （或只有cd）切换到当前用户主目录(home底下以用户名命名的文件夹)
```

## `pwd`

print working directory, 打印当前工作目录

## mkdir

```sh
mkdir 创建目录
mkdir 目录名 -p 递归创建目录
```
